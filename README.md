# Joshua Crisologo — Personal Website

Personal portfolio site built with React, TypeScript, and Vite.

## Stack

- React + TypeScript
- Vite
- CSS (no component library)

## Development

```bash
npm install
npm run dev
```

## Adding Content

- **Experience** — edit `EXPERIENCES` array in `src/App.tsx`
- **Projects** — edit `PROJECTS` array in `src/App.tsx`
- **Blog posts** — add a file to `src/blog/posts/` and import it in `src/blog/index.ts`
